:After + < span > + @keyframe

1. < span > É uma tag inline do HTML, usada para envolver trechos de texto ou conteúdo que você quer estilizar sem quebrar o fluxo do documento. Serve como o “alvo” do pseudo-elemento ::after. O span não cria nada visual por si só, mas permite adicionar efeitos extras depois do texto que ele contém.

2. ::after É um pseudo-elemento CSS que cria conteúdo virtual após o conteúdo real do elemento (no caso, o < span >).Adiciona um símbolo, ícone ou animação depois do texto do < span >. Por exemplo, você pode colocar um emoji ou uma estrela brilhante que não existe no HTML original.Importante: precisa sempre da propriedade content para funcionar.

3. @keyframes É uma regra do CSS usada para criar animações, definindo etapas do início ao fim da animação (keyframes).Anima o pseudo-elemento ::after para que ele pisque, se mova, ou mude de opacidade/cor, criando um efeito visual dinâmico junto ao < span >.
